# Loan Prediction Machine Learning Project

## Overview

This project predicts whether a loan application will be approved or not using Machine Learning. The dataset is taken from the Analytics Vidhya Loan Prediction Hackathon.

## Problem Statement

Financial institutions need to decide whether a loan should be approved based on applicant details such as income, credit history, education, and property area. The goal of this project is to build a machine learning model that predicts the loan approval status.

## Dataset

The dataset contains information about loan applicants such as:

* ApplicantIncome
* CoapplicantIncome
* LoanAmount
* Loan_Amount_Term
* Credit_History
* Gender
* Married
* Dependents
* Education
* Self_Employed
* Property_Area

Target Variable:
Loan_Status (Loan Approved or Not Approved)

## Steps Performed

1. Data loading and exploration
2. Handling missing values
3. Encoding categorical variables
4. Feature selection
5. Model training using Random Forest
6. Prediction on test dataset
7. Submission file creation

## Model Used

Random Forest Classifier (Scikit-learn)

## Result

The model achieved an accuracy score of approximately **0.79** on the Analytics Vidhya leaderboard.
